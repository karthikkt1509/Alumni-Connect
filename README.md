# Alumni Connect – Web Application

Alumni Connect is a web-based platform designed to bridge the gap between alumni and current students of an institution. 
The application provides a centralized space where alumni and students can connect, interact, and share opportunities 
such as career guidance, mentorship, and professional experiences.

## Objective
The main objective of this project is to strengthen alumni–student relationships by providing an online platform that 
encourages networking, communication, and knowledge sharing.

## Features
- User authentication and authorization
- Separate roles for alumni and students
- Custom user profiles with profile picture upload
- Alumni profile browsing and search functionality
- Secure and scalable backend architecture
- Responsive and user-friendly interface

## Tech Stack
- Backend: Django, Python
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Media Handling: Django ImageField (Pillow)
- Tools: Git, GitHub
